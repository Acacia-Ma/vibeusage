VibeUsage Webhook Status

VibeUsage does not currently expose outbound webhooks. Integrations should use scoped read endpoints, sync heartbeat state, and the MCP/OpenAPI resources listed here.

  • For current usage state, call user-scoped usage analytics endpoints with usage:read.
  • For sync freshness, use hourly usage sync fields and POST /functions/vibeusage-sync-ping.
  • Agents should treat this page as the canonical webhook capability statement until a webhook API appears in OpenAPI.
